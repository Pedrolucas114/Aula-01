Usar o terminal para linkar o VSCODE com o GIT

"git init" TEM NO GITHUB (CÓDIGO)
- Utilizei para dar start no git na pasta "Aulas" no pc (iniciada no VSCODE)

"ls" 
- Utilizei para localizar todos os arquivos/pastas presentes na Pas aula

"cd aula 01"
- Utilizei para escolher a pasta aula 01 presente dentro de aulas

"git status"
- Utilizado para mostrar todos os arquivos que sofreram alterações

"git add ." TEM NO GITHUB (CÓDIGO)
- Pega todos os arquivos alterados e adiciona em determinado local que irá salvar os arquivos para fazer outros processos

"git commit -m "First Commit"" TEM NO GITHUB (CÓDIGO)
- Add todos os arquivos alterados para First Commit
- saber que mestá mechendo e qual conta está sendo utilizada
